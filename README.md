## How to Use

To use our colors, simply add the following code to your project's CSS file:

```css
@import "@flexful-ui/colors";
```

```css
/* Now you can use the colors in your CSS */
.element {
  background-color: var(--red-50);
  color: var(--blue-500);
}
```
